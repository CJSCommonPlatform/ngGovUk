Tables

Consider putting content into tables to make it easier to scan.

Numeric tabular data

  • when comparing rows of numbers, align numbers to the right in table cells
  • use the numeric class with elements containing numbers, in order to allow numbers of the same width to be more easily compared

Collapsible table

To make rows inside a table collapsible give the table a class of collapsible, give the heading rows a class of collapsible-row-header and finally give the content rows a class of collapsible-row-content. Use ng-click and ng-show to hide and show rows.

Sortable table

Allows the table to be sortable by heading, add a class of sortable. By default it is a down arrow, add a class of sort-active-asc to change it to an up arrow
